**Ответы на вопросы на оценку 5!***

Вопросы:
1)	Разница ссылочных типов и типов значений.
2)	Что такое перегрузка методов?
3)	Как вы понимаете понятие наследование?
4)	Что означают ключевые слова virtual и override?
5)	Что такое делегат и зачем он нужен?
6)	Что такое Лямбда-выражение?
7)	Что такое LINQ?
8)	Что такое интерфейс и для чего он нужен?

Ответы: 
1) Ссылочные типы - это типы данных, которые содержат ссылки на объекты в памяти компьютера, а типы значений - это типы данных, которые хранят значения прямо в их переменных.
2) Перегрузка методов - это создание нескольких методов с одним именем, но разными параметрами. Это позволяет облегчить использование метода и делает его более гибким.
3) Наследование - это механизм в объектно-ориентированном программировании, позволяющий создавать новый класс на основе существующего класса, чтобы наследовать его свойства и методы.
4) Ключевое слово virtual используется, чтобы определить метод как виртуальный метод, который может быть переопределен в производном классе. Ключевое слово override используется в производном классе для переопределения метода, который был определен в базовом классе как виртуальный.
5) Делегат - это тип данных, который представляет ссылку на метод. Он позволяет передавать методы в качестве параметров и использовать их для обратного вызова.
6) Лямбда-выражение - это сокращенный синтаксис для создания анонимных методов. Они позволяют определить метод внутри блока кода без явного указания имени метода или его возвращаемого типа.
7) LINQ (Language Integrated Query) - это технология, позволяющая выполнять запросы к различным источниками данных, включая базы данных, коллекции и массивы, используя общий язык запросов.
8) Интерфейс - это тип данных, который определяет набор методов и свойств, которые должны быть реализованы в классе, который реализует этот интерфейс. Он позволяет создавать более гибкое и модульное приложение, разделяя реализацию классов и их поведение.
